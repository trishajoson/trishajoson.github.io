Reviews on the game

Jay Davis

This game is incredible. Valorant has the same great gunplay as CS, but with mechanics that make it so that the game never gets stale. This game succeeds in certain areas that CS did not, and introduces smaller mechanics that CS should've had from the get go, for example, the selling system, requesting system, and the pinging system. The different characters introduce a new level of teamplay, with characters that can use their abilities in ways that synergize with others, allowing for great plays to be made. Another big factor about this game is the fact that it can run on potato pcs. a low end pc can run this game smoothly, with little to no problems appearing. Unlike valve, Riot actually listens to their community, and with every patch the game gets better and better. I personally found CS:GO to get stale after a while, so this game was a breath of fresh air for me and my friends. I totally suggest this game to anyone who is interested. GIVE IT A SHOT!
